## Content

The reconstruction of Figure 2 from the paper is divided across three notebooks:

- `benchmark_kang_analysis.ipynb` reproduces subfigures A, B, C, and F.
- `benchmark_kang_metrics.ipynb` reproduces subfigure D.
- `benchmark_synthetic.ipynb` reproduces subfigure E.

## Installation

To run the notebooks, please install the following packages:

```shell
pip install scSpectra scarches scikit-misc
```
